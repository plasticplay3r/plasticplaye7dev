Overview

End UI Overhaul is a complete redesign of Minecraft’s interface inspired by the mysterious aesthetic of the End. This pack replaces menus, buttons, icons, and UI elements with a clean, purple‐themed style while remaining fully compatible with the entire Minecraft 1.21 update family.

Features

  • Fully redesigned End‐themed UI
  • Custom buttons, menus, and inventory textures
  • Clean purple aesthetic inspired by the End
  • Compatible with all 1.21.x versions and snapshots
  • Lightweight and performance‐friendly
  • Works with vanilla and modded clients

Supported Versions

  • 1.21
  • 1.21.1 → 1.21.11
  • 1.21.2 pre‐releases
  • 24w33a → 24w46a snapshots

Installation

  1. Download the .zip file
  2. Open Minecraft
  3. Go to Options → Resource Packs
  4. Click Open Pack Folder
  5. Drop the .zip inside
  6. Enable the pack
